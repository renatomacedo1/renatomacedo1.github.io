WebSIG Platform

The project reported here is a web application that uses the Leaflet API to mark and display points, lines, and polygons, as well as to associate them with user-created layers. To represent spatial information, a PostgresSQL database with PostGIS extension was used to use the geometry field. Communication with the database is done through AJAX requests that communicate with web services developed in PHP. Each point, line, or polygon is associated with the user who created it. Thus, to access the application, the user needs to register and authenticate.
Technologies
- HTML5, CSS3, Bootstrap, PHP, JavaScript, JQuery e PostgreSQL
